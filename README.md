# 📊 Ecommerce Sales Dashboard | Power BI
# 📌 Project Overview

This project is an interactive Ecommerce Sales Dashboard built using Power BI.  
The dashboard provides valuable business insights by analyzing sales performance, profit trends, customer behavior, and payment methods.

It helps businesses track KPIs and make data-driven decisions through interactive visualizations.

---

# 🚀 Key Features

✅ Sales Performance Analysis  
✅ Profit Tracking Dashboard  
✅ Quantity Analysis  
✅ State-wise Sales Insights  
✅ Monthly Profit Trends  
✅ Payment Mode Analysis  
✅ Sub-category Performance Tracking  
✅ Interactive Filters & Slicers  
✅ Dark Theme Professional UI

# 📈 Dashboard Insights

- Maharashtra recorded the highest sales.
- COD was the most preferred payment method.
- Clothing category had the highest quantity sold.
- Monthly profit trends helped identify business performance patterns.
